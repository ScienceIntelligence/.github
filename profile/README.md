# OpenScientists

**OpenScientists** is a public-interest initiative to build open infrastructure for AI-assisted scientific discovery.

Scientific progress is upstream of advances in energy, medicine, materials, computation, and climate. If AI becomes part of the machinery of discovery, that machinery should remain **open, inspectable, contestable, and reproducible**.

## What we are building

OpenScientists is organized around five connected efforts:

1. **Tacit knowledge library**  
   A structured, versioned repository of scientific heuristics, debugging instincts, failure modes, and practical judgment that rarely appears in papers.

2. **Agent runtime system**  
   An execution environment for AI-assisted research, including literature search, code execution, tool use, persistent memory, and citation tracing.

3. **Scientific ability benchmark**  
   Evaluation based on real research tasks such as hypothesis generation, confounder detection, ablation design, methodological critique, and uncertainty calibration.

4. **Model development**  
   Open model improvement for scientific reasoning, emphasizing calibrated uncertainty, disciplined tool use, and long-horizon planning rather than fluent text alone.

5. **Metascience and governance**  
   Empirical study of how these systems affect scientific production, reviewer burden, citation networks, publication quality, and institutional norms.

## Why now

The window for building open scientific AI infrastructure is time-sensitive. Once institutions and researchers standardize on closed systems, the cost of building a credible open alternative rises sharply.

The question is not whether AI will enter science, but whether the governing infrastructure will remain a public good.
